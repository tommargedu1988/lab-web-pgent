# lab-web-pgent

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为 HTML 静态页面的存放与展示空间。

## 项目简介

`lab-web-pgent` 旨在集中管理一系列独立的 HTML 页面。这些页面可能包含实验性设计、原型、演示或信息展示内容，适合快速浏览与分享。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 通用资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立的页面，可直接通过浏览器打开或通过 GitHub Pages 访问。
- `assets/`：存放页面可能共享的样式表、脚本或图片资源。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，无跨页面交互逻辑。
- 每个页面可包含自己的样式与脚本，也可引用 `assets/` 中的公共资源。

## 维护说明

- 如需添加新页面，请在 `pages/` 目录下创建新的 HTML 文件，并确保其路径正确。
- 更新或删除页面时，请同步更新入口文件（如有）中的链接。
- 建议为每个页面添加清晰的注释或说明，便于后续维护。

## 许可

本仓库默认不附带特定许可证，内容仅供学习与参考。如需引用或分发，请自行确认合规性。
